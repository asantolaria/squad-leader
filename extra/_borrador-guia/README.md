# Borrador de la guía didáctica (no publicado)

Capítulos didácticos originales (01–09 y glosario) escritos en su día para una guía
"aprende a jugar" capítulo a capítulo. **No se publican**: el reglamento traducido ya
cubre esas reglas, así que en la web solo quedan las secciones que aportan algo nuevo
(solitario, escenarios de práctica, referencia rápida y recursos), en `extra/`.

Se conservan aquí por si se quieren reaprovechar. Esta carpeta está excluida del build
de MkDocs (`exclude_docs: _borrador-guia/` en `mkdocs.yml`).
