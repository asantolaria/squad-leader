## Secuencia del turno (memorízala)

```
TURNO DE JUGADOR (bando en fase = "A"; el otro = "B")
 1. REAGRUPAMIENTO ...... A intenta recomponer unidades rotas (con líderes)
 2. FUEGO PREPARADO ..... A dispara a plena potencia → esas unidades NO mueven
 3. MOVIMIENTO .......... A mueve una unidad/pila completa cada vez
 4. FUEGO DEFENSIVO ..... B dispara reaccionando al movimiento de A
 5. FUEGO DE AVANCE ..... A (los que movieron) disparan a MEDIA potencia
 6. DESBANDADA .......... unidades rotas HUYEN hacia cobertura/retaguardia
 7. AVANCE .............. A mueve 1 hex SIN provocar fuego (último salto / contacto)
 8. CUERPO A CUERPO ..... se resuelven los hexes con unidades de ambos bandos
```

Luego juega B con las mismas 8 fases; después avanza el marcador de turno.

---

## La decisión clave de cada unidad, cada turno

```
¿DISPARAR o MOVER?
 ├─ Disparar en PREP FIRE  → plena potencia, pero te quedas quieto
 ├─ Mover y NO disparar    → maniobra, pero te expones al fuego defensivo
 └─ Mover y disparar en AVANCE → media potencia (fuego y movimiento)
```

## Procedimiento de FUEGO de infantería (10.3)

```
1. ¿Hay LÍNEA DE VISIÓN?            (si la corta un obstáculo → no disparas)
2. SUMA la potencia de fuego (FP)   (combina varias unidades del mismo hex)
3. MULTIPLICADORES de FP:           ×2 adyacente · ×½ larga distancia
                                    ×½ atacante en movimiento · ×½ fuego de área
                                    (más del DOBLE de alcance → no disparas)
4. Localiza la COLUMNA de FP        (en tu Tabla de Fuego, con la FP ya multiplicada)
5. MODIFICADORES a la TIRADA (11.1): cobertura del objetivo = + (peor para el tirador)
                                    blanco moviéndose al descubierto = − (mejor)
6. Tira 2d6 ± DRM → lee resultado:  — nada · M chequeo · nº chequeo agravado · KIA baja
```

> Recuerda la diferencia: **alcance/adyacencia/movimiento/área → multiplican la POTENCIA**
> (te cambian de columna); **el terreno → modifica la TIRADA de dados**.

## Procedimiento de CHEQUEO DE MORAL

```
1. El objetivo tira 2d6 (± modificadores: líder ayuda, terreno expuesto perjudica)
2. Compara con su MORAL (3er número de la ficha)
   ≤ moral → AGUANTA (a veces queda "pinned" este turno)
   > moral → SE ROMPE (cara rota; tendrá que huir)
   > moral por mucho → ELIMINADA
```

## Procedimiento de COMBATE DE BLINDADOS

```
1. IMPACTO  → tabla "to hit": distancia + modificadores (parado/en marcha, blanco móvil…)
2. PENETRA  → penetración del proyectil vs. blindaje del LADO golpeado (frente/costado/trasera)
              (a menos distancia, más penetración)
3. DAÑO     → destruido/ardiendo · inmovilizado · arma destruida · tripulación afectada
```

## Recordatorios que más se olvidan

- [ ] **Una unidad/pila se mueve entera** antes de mover la siguiente.
- [ ] **El bando inactivo dispara** en fuego defensivo: comprueba LOS antes de cruzar.
- [ ] **Prep fire = no mover; mover = solo fuego de avance** (media potencia).
- [ ] La **fase de avance (1 hex) NO provoca fuego**: úsala para el último salto a cubierto.
- [ ] **Combina fuego** sobre un objetivo en vez de repartir disparos flojos.
- [ ] **Reagrupa** cada turno: lleva un líder a la retaguardia.
- [ ] **Rodear** al enemigo impide que sus rotos huyan → se autodestruyen.
- [ ] **Carros + infantería juntos**; un carro solo en terreno cerrado muere.
- [ ] Dispara a los carros por el **costado/trasera**, **parado** y **de cerca**.
- [ ] **Humo** para cruzar abierto; **lanzallamas/cargas** para edificios y búnkeres.
- [ ] **Protege a líderes y dotaciones de cañón.**
- [ ] No **apiles** en terreno claro: un disparo afecta a todo el hex.
